## 🔄 完整数据流

```
1. 用户发起请求
   │
   ▼
2. OpenClaw 接收指令
   │
   ▼
3. SSH 连接到 Mac
   │
   ▼
4. Mac 执行转录脚本
   │
   ├─► 下载音频 (yt-dlp)
   │    │
   │    ▼
   │    YouTube → /tmp/audio.webm
   │
   ├─► 转录音频 (Whisper)
   │    │
   │    ▼
   │    /tmp/audio.webm → transcript[]
   │    │
   │    └─► 每 5 分钟更新进度
   │         │
   │         ▼
   │         /tmp/transcribe_progress.log
   │
   └─► 生成总结
        │
        ▼
        /tmp/transcribe_result.json
        /tmp/transcript_*.txt

5. 结果返回给 OpenClaw
   │
   ▼
6. 用户查看结果
```

## 📡 网络通信细节

### NAS → Mac SSH 连接

```bash
# 连接命令
ssh cc@192.168.1.y "python3 /tmp/transcribe_background.py 'URL' zh"

# 连接详情
- 源 IP: 192.168.1.x (NAS)
- 目标 IP: 192.168.1.y (Mac)
- 端口：22 (SSH)
- 协议：SSH-2.0
- 认证：Public Key / Password
```

### Mac → YouTube 下载

```bash
# 下载命令
python3 -m yt_dlp -f bestaudio -o /tmp/audio.webm "URL"

# 连接详情
- 源 IP: 192.168.1.y (Mac)
- 目标：YouTube CDN
- 端口：443 (HTTPS)
- 协议：HTTPS/TLS
```

## 🎯 部署配置

### NAS 端配置

```json
// ~/.openclaw/openclaw.json
{
  "agents": {
    "defaults": {
      "timeoutSeconds": 3600
    }
  },
  "tools": {
    "timeout": 1800
  }
}
```

### Mac 端配置

```bash
# 安装依赖
pip3 install yt-dlp faster-whisper

# 复制脚本
scp /tmp/transcribe_background.py cc@mac:/tmp/
scp /tmp/start_transcribe.sh cc@mac:/tmp/

# 添加执行权限
chmod +x /tmp/start_transcribe.sh
chmod +x /tmp/transcribe_background.py
```
